Advanced Usage Example

This example shows how to use the promise-based API to handle zap results programmatically.

// Using promise-based API: async function handleAdvancedDonation() { try { const result = await BuzzZapModal.openAsync('npub1...', { amount: 5000, method: 'lightning' }); if (result.status === 'ok') { // Handle successful payment console.log('Payment successful:', result); // Update UI document.getElementById('result-container').innerHTML = 'Payment successful! ID: ' + result.payment.id; document.getElementById('result-container').style.display = 'block'; } } catch (error) { console.error('Payment failed or was cancelled'); document.getElementById('result-container').innerHTML = 'Payment failed or was cancelled'; document.getElementById('result-container').style.display = 'block'; } }

Customization Example

The modal can be customized to match your site's branding. This example shows a custom colored modal.

// Custom styling: function openCustomZapModal() { // First configure the modal appearance BuzzZapModal.config({ primaryColor: '#ff6b6b', logo: 'https://nostr.buzz/logo.png', i18n: { title: 'Support this project', payButton: 'Send Support', } }); // Then open the modal BuzzZapModal.open('npub1tapj48eekk8lzvhupfxg4ugdgthaj97cqahk3wml97g76l20dfqspmpjyp', { amount: 4200, comment: 'Custom styled zap' }); }
